Accel GUI - является графическим интерфейсом популярного сервера доступа accel-ppp.

Приложение получает данные от сервера путем отсылки соответствующих команд по сети, использую протокол tcp, на порт, указанный в файле конфигурации vpn сервера.

На данный момент в приложении реализован не весь возможный функционал, но недоработки активно устраняются в процессе разработки.

На данный момент приложение поддерживает следующие возможности:

  1. Многоязычность
    Поддержка двух языков: английского и русского. Переключение осуществляется автоматически.
  2. Мультисервер
    Получение данных одновременно с нескольких серверов.
  3. Перезагрузка
    Выполнение перезагрузки файла конфигурации.
  4. Отключение
    Отключение службы сервера, а также отмена этого действия.
  5. Перезапуск
    Перезапуск службы сервера.
  6. Фильтр
    Фильтрация сессий по содержимому одного из четырех столбцов.
  7. Сортировка
    Сортировка данных по различным столбцам.
  8. Автозапуск
    Активация автоматического открытия окна сервера при запуске приложения.
  9. Сброс
    Обрыв текущей или всех сессий.
  10. Авторизация
    Поддержка трех типов учетных записей.
Внимание! Более детальное описание всех функциональных возможностей приложения можно найти перейдя по ссылке "Настройка" в главном меню справочной системы.
