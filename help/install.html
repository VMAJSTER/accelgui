Установка

Подробная инструкция по установке приложения

Так как в приложении нет механизма проверки и последующей автоматической загрузки и установки последней версии, лучшим способом установки является клонирование репозитория с github. Для этого нужно установить как саму среду разработки, для последующего корректного запуска приложения, так и пакеты для работы с git репозиториями и прочие. Процесс установки и запуска приложения будет показан на примере дистрибутива Ubuntu.

  1. Установим необходимые пакеты
    sudo apt-get install gambas3 git gksu -y

  2. Создаем каталог для будущего приложения
    mkdir -p ~/soft/

  3. Переходим в созданный ранее каталог
    cd ~/soft

  4. Клонируем репозиторий
    git clone https://github.com/ZagirovAA/accelgui.git

  5. Назначаем файлу приложения бит исполнения
    chmod +x ~/soft/accelgui/accelgui.gambas

  6. После клонирования появится каталог accelgui, в котором будет исполняемый файл accelgui.gambas, ярлык для которого нужно создать на рабочем столе, использую средства окружения рабочего стола, которое используете вы.

  7. Для первоначальной настройки приложения необходимо запустить его с правами root, для чего можно использовать инструмент gksu. После добавления имеющихся у вас серверов и смены паролей для всех трех учетных записей, можно запустить приложение в нормальном режиме и начать им пользоваться.

Внимание! Более детальное описание всех функциональных возможностей приложения можно найти перейдя по ссылке "Настройка" в главном меню справочной системы.
